A person who codes Web pages and server programs. Web programmers are typically thoroughly familiar with HTML and XML and some number of languages, including JavaScript, VBScript, ASP, Java, Python, Perl, C/C++ and others. They are also familiar with the development tools that pertain to their particular Windows, Mac or Linux computing environment.
